🔹 **Title**

Cloud-Based Financial Data Pipeline & Analysis

🔹 **Overview**

This project demonstrates an end-to-end data analysis workflow using cloud technologies and visualization tools. The goal is to extract meaningful insights from financial transaction data.

🔹 **Tech Stack**

» Python (Pandas)

» Amazon Web Services (S3)

» Google BigQuery

» SQL

» Power BI


🔹 **Data Pipeline**

CSV → Python (cleaning) → S3 → BigQuery → SQL → Power BI

🔹 **Key Insights**

» Identified transaction trends across months

» Analyzed customer spending behavior

» Compared transaction distribution across channels

🔹 **Project Workflow**

» Cleaned raw data using Python (Pandas)

» Uploaded dataset to AWS S3

» Performed SQL analysis using BigQuery

» Built interactive dashboard using Power BI
